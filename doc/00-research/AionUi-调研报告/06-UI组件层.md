## 四、UI 规范对比

### 4.1 组件库

| 项目 | UI 框架 | 图标库 |
|------|---------|--------|
| AionUi | Arco Design | @icon-park/react |
| tech-cc-hub | Radix UI | 无特定图标库 |

### 4.2 样式方案

| 项目 | 方案 |
|------|------|
| AionUi | UnoCSS + CSS Modules |
| tech-cc-hub | Tailwind CSS v4 |

### 4.3 设计亮点

**IconPark HOC**:
```typescript
// 统一处理 IconPark 组件
const IconParkHOC = (Component: React.ComponentType<IconProps>) => {
  return (props: IconProps) => {
    return (
      <Tooltip content={props.title}>
        <Component {...props} />
      </Tooltip>
    );
  };
};
```

**ShimmerText 动画**:
```typescript
// 闪烁加载效果
const ShimmerText: React.FC<{ text: string; loading?: boolean }> = ({ text, loading }) => {
  return (
    <span className={loading ? 'shimmer' : ''}>
      {text}
    </span>
  );
};
```

## 五、tech-cc-hub 借鉴建议

### 5.1 可直接借鉴

| 功能 | AionUi 实现 | 移植价值 |
|------|-------------|----------|
| **SlashCommandMenu** | 完整的命令菜单 | 高 |

| **Markdown 分块** | 代码高亮分块 | 中 |

### 5.2 差异说明

由于 UI 框架不同（AionUi 使用 Arco Design + UnoCSS，tech-cc-hub 使用 Radix + Tailwind），大部分组件需要重新实现。

### 5.3 核心组件适配

```typescript
// tech-cc-hub 版本的 SlashCommandMenu
interface SlashCommandMenuProps {
  commands: SlashCommandItem[];
  onSelect: (command: SlashCommandItem) => void;
  open: boolean;
}

// 使用 Radix Popover + Tailwind 实现
export const SlashCommandMenu: React.FC<SlashCommandMenuProps> = ({ commands, onSelect, open }) => {
  return (
    <Popover open={open}>
      <Popover.Content>
        {commands.map(cmd => (
          <CommandItem key={cmd.name} onSelect={() => onSelect(cmd)}>
            <span className="font-mono">/{cmd.name}</span>
            <span className="text-muted">{cmd.description}</span>
          </CommandItem>
        ))}
      </Popover.Content>
    </Popover>
  );
};
```
