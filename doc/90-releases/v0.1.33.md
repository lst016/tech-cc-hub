# v0.1.33 版本更新说明

## 更新内容

- 增强 BrowserWorkbench 标注体验：悬停卡片展示元素尺寸和关键样式，标注面板支持更紧凑的定位、样式编辑和 CSS 片段记录。
- 优化运行时工具面：标准会话默认保留 BrowserView 工具，并提示优先复用当前 BrowserView，减少重复启动本地页面。
- 扩展 CodeGraph 优先检索约束：覆盖 `Search`、常见 shell 搜索命令和未被用户点名的源码 `Read`，减少跳过 CodeGraph 的宽泛探索。
- 会话每轮结束后自动触发当前工作区 CodeGraph 增量同步，避免新保存的符号长期停留在旧索引里。
- 优化前端启动体验和首屏加载：增加静态启动骨架，并对较重面板做懒加载 fallback。
- 版本号升级到 `0.1.33`。

## 验证

- `npx eslint`（变更文件范围，0 errors）
- `npm run build`
- `npx tsc --project test/electron/tsconfig.json`
- `node --test dist-test/test/electron/*.test.js`
- `npm run package:win`
