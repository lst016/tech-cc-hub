# v0.1.51 版本更新说明

## 更新内容

- 聊天选区浮层现在同时支持“添加到对话”和“评论”，可以把选中的助手内容按条挂评论后逐个回复，也支持直接发送评论。
- 修复进行中的 workflow 会话被 `session.continue` 打断的问题；当会话仍有活跃 workflow run 时，新的追问会直接 append 到当前 runner。
- 提升选区评论交互稳定性：跨段选区时优先锚定起始块，popover 聚焦期间不会被 `selectionchange` 误清空。
- Dev Bridge 在开发环境替换 fallback shim 后会重新订阅 IPC 事件，避免 dev 环境聊天记录/事件丢失。
- Figma 配置与报错提示更稳：保留官方 MCP 配置、补充 PAT/MCP 并存提示，并更新工作流提示词约束。
- 新增聊天选区评论 smoke 脚本与针对性回归测试，覆盖评论引用、去重、活跃 workflow 追问和 dev bridge 重连。
