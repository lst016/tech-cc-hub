# v0.1.30 版本更新说明

## 更新内容

- 修复会话内切换模型只更新前端状态的问题：新增 `session.set_model` IPC，切换后会同步写回会话存储并广播最新 `session.status`。
- 修复 Custom Gateway 下 `DeepSeek-V4-Pro` 被错误归一化为 `deepseek-v4-pro` 的问题：自定义网关保留模型大小写，只有官方 DeepSeek provider 会做小写归一化。
- 模型选择器会对 DeepSeek 大小写别名做去重；旧会话里保存的小写别名会自动映射回当前配置里的模型名大小写。
- 浏览器预览 shim 同步支持会话模型状态，避免 dev/preview 和 Electron 运行时类型不一致。
- 版本号升级到 `0.1.30`。
