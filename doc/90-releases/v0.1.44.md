# v0.1.44 版本更新说明

## 更新内容

- 输入区的进行中目标条新增关闭按钮，可直接收起当前目标展示，避免目标状态像被锁定在输入框上方。
- 目标条改为更轻量的状态条，用绿色和琥珀色状态点区分 active / blocked，降低高度、阴影和视觉占用。
- 删除待发送队列里“运行中可点「插入」作为补充命令；空闲后会自动续发。”这句冗余说明。
- 合并 v0.1.43 之后的工作台上下文增强：BrowserView 登录态优先、Chrome Cookie 同步、多格式附件解析、Preview 首个变更跳转、CodeGraph / Runtime 工具面收敛和 Windows 图标打包加固。
