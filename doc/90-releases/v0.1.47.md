# v0.1.47 版本更新说明

## 更新内容

- 跟进 Claude Agent SDK `0.3.187` 运行时能力：补充 TaskCreate/TaskUpdate/Plan 工具迁移、`tool_use_meta` 消费、result origin 与拒绝/过载错误处理。
- 增强运行时配置与上下文接入：支持结构化 sandbox settings、linked workspace `additionalDirectories`、`getContextUsage` 上下文用量读取，以及 startup CLI 子进程预热。
- 补齐 Workflow 与团队协作事件渲染：适配 Task 工具事件卡片，新增 ConfigChange、TeammateIdle、TaskCompleted、MessageDisplay、PreCompact 等 hook 注册。
- 收敛团队协作 hook 行为：TeammateIdle 与 TaskCompleted 只记录运行日志，不再向模型追加额外上下文，减少重复提醒对后续执行的干扰。
- 优化 Windows 更新与打包链路：补强 updater fallback、release workflow 与 Windows installer 元数据校验，降低发布资产缺失或元数据不一致的风险。
- 清理进程组卡片残留常量，延续长历史渲染优化后的轻量组件路径。

## 验证
