# v0.1.52 版本更新说明

## 更新内容

- 新增会话重命名能力：侧边栏会话菜单现在支持直接修改会话标题，便于整理和继续跟进历史会话。
- 优化 `session.continue` 的延续链路：继续会话时统一重建 stateless continuation payload，并保留更稳定的附件/上下文恢复行为。
- 修复聊天选区作用域：拖选评论时会把可选区限制在当前消息正文内，减少跨卡片误选和 popover 干扰。

## 验证

- `npm run transpile:electron`
- `npm run build`
- `npm run test:electron:build`
- `npm run package:win`

## 注意

- 本次版本基于已发布的 `v0.1.51` 继续追加 patch 版本，不复用旧 tag。
- 全量 `npm run lint` 仍有仓库既有噪音，本次发布以前述定向构建与打包结果为准。
