# v0.1.42 版本更新说明

## 更新内容

- 修复 CodeGraph 优先检索 hook 误拦截明确文件 `Read` 的问题，避免 `Read` 被替换为提示信息后导致 `Edit` / `Write` 一直报 `File has not been read yet`。
- 保留 CodeGraph 对 `Grep`、`Glob`、`Search`、`Task` 和宽泛 shell 搜索的优先检索约束，避免回退到无边界源码探索。
- 修复文本文件 `Read` 在配置图片预处理模型时被误送入图片摘要链路的风险；现在只有 raster 图片扩展名才触发图片 Read 预处理。
- 版本号升级到 `0.1.42`。
