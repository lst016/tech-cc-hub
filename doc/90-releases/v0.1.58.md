# v0.1.58 版本更新说明

## 更新内容

- 侧边 BTW 会话升级为可持续的多轮后台会话：补齐独立 session 生命周期、切换与关闭隔离，以及后台完成/异常状态提示，主会话不会再被侧聊替换。
- 新增聊天图片能力：支持图片生成参数面板、图片附件画廊和生成上下文透传，可按画幅、2K/4K 分辨率及数量组织图片生成请求。
- 新增 ECharts 消息卡片：可在回复中安全解析并渲染 `:::echarts` 图表块，支持折线图、柱状图和饼图切换，同时限制不安全 HTML、链接及配置字段。
- 优化提示词编辑体验：拆分输入区控制组件，完善原生撤销、草稿恢复、附件清理、队列发送和 Markdown 选区定位，降低复杂交互之间的状态干扰。
- 重构侧边栏工作区与会话组件，统一工作区导航、搜索、折叠及会话操作，并补充计划完成门禁和会话状态回归覆盖。
- 增强 Codex 代理流式输出、图片生成上下文和 Canvas 工作区插件桥接，减少空成功、流式事件及插件状态同步中的边界问题。
