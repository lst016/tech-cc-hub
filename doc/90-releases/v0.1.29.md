# v0.1.29 版本更新说明

## 更新内容

- 知识图谱默认切换到 CodeGraph：托管索引放在工作区 `.tech/codegraph`，检索前会自动初始化并执行增量同步，不依赖 LLM 或向量服务。
- 移除 RepoWiki 主链路：砍掉 RepoWiki UI tab、旧知识库生成/向量索引代码、RepoWiki QA 脚本和 `third_party/repowiki` 打包依赖。
- Agent 检索提示改为优先使用 CodeGraph：代码证据场景会鼓励先用 `codegraph_search` / `codegraph_context`，再按需回退到文件读取和全文搜索。
- Browser Workbench 增强 fetch 捕获：内置 browser MCP 现在可以读取 Fetch/XHR 请求、响应和 body，方便会话里直接拿接口数据做分析。
- 修复内置 MCP 复用导致的 `Not connected`：Figma、Browser、Knowledge、Plan、IDEA、Cron、Admin 等内置 SDK MCP 每次 Agent run 都创建新实例。
- 改善输入和滚动体验：长 JSON 粘贴按纯文本处理并收在输入框内，模型下拉框不再被输入框容器裁剪，底部状态下点击“到顶部”不会被自动滚回底部。
- 保留外部/桌面 MCP 配置入口：设置页继续支持配置外部 MCP，Figma Desktop MCP 可按本机服务接入。
