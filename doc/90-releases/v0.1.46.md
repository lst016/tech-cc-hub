# v0.1.46 版本更新说明

## 更新内容

- 移除会话消息在渲染侧的 600 条上限，`useAppStore` 保留完整消息数组，避免长会话回看时丢失上下文。
- 优化聊天主视图渲染路径：去掉 `displayMessages` 中间层，直接基于可见消息构建渲染条目，减少大历史会话下的重复计算。
- `useMessageWindow` 的历史窗口按 `user_prompt` 边界对齐，加载更多历史时避免切开同一轮对话。
- `ProcessGroupCard` 增加 `React.memo`，并拆分变更文件行与预览弹层组件，降低父组件刷新时的子树重建成本。
- 兼容可远端 resume 的 provider：对支持远端恢复的运行时保留远端续跑路径，对需要压缩的历史继续使用本地无状态压缩。
- 补充长历史窗口、进程组布局、预览刷新、浏览器工作台可见性等回归覆盖，锁定本轮性能与兼容性行为。
