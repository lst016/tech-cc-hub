# v0.1.32 版本更新说明

## 更新内容

- 修复用户消息气泡把内部 `<file_references>` 结构块原样展示出来的问题：正文现在只显示用户实际输入。
- 文件引用和消息引用会渲染成独立上下文卡片，样式与浏览器标注、代码引用保持一致，不再显示协议 JSON。
- 修改用户消息时会保留 `browser_annotations`、`code_references`、`file_references`、`message_references` 四类隐藏上下文。
- 版本号升级到 `0.1.32`。
