# v0.1.59 版本更新说明

## 修复内容

- 修复 Windows 安装包启动时偶发的 Electron 主进程报错：显式将 `better-sqlite3` 及其原生运行时纳入 `app.asar.unpacked`，避免仅依赖 electron-builder 自动推断导致包入口缺失。
- 加强安装包启动检查：packaged smoke 现在同时识别 `Cannot find module` 和 `Cannot find package`，可以拦截本次截图中的 ESM 包解析失败。
- Release 流水线在上传 Windows 产物前会实际启动 `win-unpacked` 进行冒烟检查，启动失败时不再发布安装包。
- GitHub Release 现在会按 tag 自动读取仓库内对应的中文版本说明，避免发布页只有自动生成的变更链接。

## 验证

- Windows 安装包依赖配置回归测试
- Electron 测试构建
- Windows packaged startup smoke
- GitHub Actions `Release` 流水线构建、启动检查与产物发布
