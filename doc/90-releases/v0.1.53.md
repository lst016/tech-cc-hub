# v0.1.53 版本更新说明

## 更新内容

- Codex 配置页新增手填兜底入口：Agent 现在只负责读取和整理 Codex credential JSON，不再自动写入 `api-config.json`。
- Codex 引导流程改为只读 handoff，明确禁止运行会改写配置的 setup 脚本，避免在别的机器上误删现有网关或模型来源。
- MCP 内置服务器默认启用项继续保持浏览器、admin、design、cron、plan、knowledge，其他内置项改为可勾选开启。

## 验证

- `npm run transpile:electron`
- `npm run build`
- `npm run test:electron:build`
- `node --test dist-test/test/electron/codex-oauth-provider.test.js`
- `node --test dist-test/test/electron/builtin-mcp-registry.test.js`
