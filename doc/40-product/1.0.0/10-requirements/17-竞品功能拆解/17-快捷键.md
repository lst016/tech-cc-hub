# 17-快捷键

## Goal
把高频操作从鼠标迁移到键盘，提高资深用户效率。

## Scope
- 窗口级快捷键
- 输入区快捷键
- 模式切换快捷键
- 终端和侧栏快捷键

## Flow
```mermaid
flowchart TD
  A[用户按下快捷键] --> B[触发面板或模式变化]
  B --> C[保留当前上下文]
```

## Detail
- 快捷键至少覆盖新建会话、聚焦输入框、打开终端、切换侧栏、切换模式。
- 快捷键文档必须和界面行为一致。

## Acceptance
1. 高频动作有一致快捷键。
1. 快捷键不会破坏当前编辑上下文。
1. 快捷键状态和 UI 状态同步。
