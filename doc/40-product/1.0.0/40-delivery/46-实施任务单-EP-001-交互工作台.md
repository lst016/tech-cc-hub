# 46-实施任务单-EP-001-交互工作台

## Purpose
把 `EP-001` 进一步拆成前后端可执行任务单，便于直接进入开发排期。

## Scope
本文件聚焦交互工作台相关任务，不覆盖 Task Graph 或治理模块。

## Behavior / Flow
### Frontend Tasks

| Task ID | Task | Depends On |
|---|---|---|
| `FE-EP001-01` | 实现 Session Sidebar 基础结构 | 无 |
| `FE-EP001-02` | 实现 Chat Workspace 顶部 Agent Picker | `FE-EP001-01` |
| `FE-EP001-03` | 实现消息流和输入框状态机 | `FE-EP001-02` |
| `FE-EP001-04` | 接入 Live Timeline 侧栏入口 | `FE-EP001-03` |
| `FE-EP001-05` | 接入 Replay / Analysis / Artifacts 跳转入口 | `FE-EP001-04` |

### Backend Tasks

| Task ID | Task | Depends On |
|---|---|---|
| `BE-EP001-01` | 提供 Session 创建 / 列表 / 恢复能力 | 无 |
| `BE-EP001-02` | 提供聊天输入与执行状态接口 | `BE-EP001-01` |
| `BE-EP001-03` | 写入 `chat_agent_selected` / `session_created` 等事件 | `BE-EP001-02` |
| `BE-EP001-04` | 提供 Session 关联的 Replay / Analysis / Artifact 概览 | `BE-EP001-02` |

### QA Tasks

| Task ID | Task | Depends On |
|---|---|---|
| `QA-EP001-01` | 验证默认 Agent 为 `Claude Code` | `FE-EP001-02`, `BE-EP001-02` |
| `QA-EP001-02` | 验证聊天 Agent 互斥切换 | `QA-EP001-01` |
| `QA-EP001-03` | 验证 Session 恢复与事件可见性 | `FE-EP001-03`, `BE-EP001-03` |
| `QA-EP001-04` | 验证聊天到 Replay / Analysis 跳转链路 | `FE-EP001-05`, `BE-EP001-04` |
